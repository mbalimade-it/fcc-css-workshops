# FreeCodeCamp CSS Workshops

This repository contains all my workshop projects from the **FreeCodeCamp Full Stack Developer curriculum**.

## Overview

Workshops are guided exercises where I learned and practiced CSS concepts. Each workshop focuses on a specific set of skills.

## How to Use

- Each folder contains the full code for that workshop.
- Open the `index.html` file in your browser to view the result.
- Screenshots and README files (if included) show what I learned in each workshop.
